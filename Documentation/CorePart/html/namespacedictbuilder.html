◆ getdict()

def dictbuilder.getdict (   filename)

Builds a dictionary containing (word : word frequency) as key-value pairs.

Parameters
filename: string - name/path of the c++ file
Return values
final_dict: python-dict - a dictionary containing words and their frequencies.

-> The comments are trimmed out from the code using comment_remover().
-> The C++ code is split into functions: A function is recognised by '(' and the function code is extracted using '{' and '}'.
-> These functions are reduced into simpler forms using reduce(), and appended to the main code for each function call made.
-> Global code which is not part of any function is appended to the main code.
-> The keywords are removed and the dictionary is built, by calculating the word count.

Reduces a c++ function in the c++ code into simpler form.

Parameters
f: string - name of the function in c++ code.
func: python-dict - dictionary containing (function name : function code) as key-value pairs.
final_func: python-dict - dictionary containing final reduced forms of the function definitions.
Returns
: void - We modify the final_func dictionary in this function.

-> Say, if in the definition of a function f(), some other function f1() is called 'c1' times, then the definition of f1() is made explicit in the definition of f() 'c1' times.
-> To tackle the problem of recursive functions, a list of the function names which were already replaced is maintained, and this list contains "main" and "f" initially.
