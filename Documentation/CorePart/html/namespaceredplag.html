  map2 
)

Takes two dictionaries as input and calculates their percentage of similarity based on the values in key-value pairs.

Parameters
map1: python-dict - Dictionary containing word frequencies of file1
map2: python_dict - Dictionary containing word frequencies of file2
Returns
: float - a measure of similarity between the two files, as a percentage.

-> Convert the dictionaries into sorted lists and pad the smaller vector with 0s.
-> The similarity between 2 files must be a property of the files i.e. should not depend on the other files in the directory.
-> The final metric for similarity is calculated as distance between the padded vectors, divided by the max of norm(vector1), norm(vector2).
(we felt distance between the vectors (distance betwen endpoints) was a better metric to measure similarity than the angle between them, and we used max( norm(vector1), norm(vector2) ) in the denominator for normalization.)

Core Function, which creates a .csv and a .png file, for interpreting the results of RedPlag Plagiarism Checker.

Parameters
inpath: string - path of directory containing the files to be checked for plagiarism.
outpath: string - path of directory to output the result files. Preferably empty.
cplusplus: bool - a boolean value informing whether the files are c++ code or simple text.
stub: bool - a boolean parameter informing whether a stub code is used.
stubpath: string - path of the stub code file (not the directory), if a stub code was given. Can be any valid string, in case no stub code is used.
Returns
: void - this function creates a .csv and a .png file in 'outpath' directory

-> Store names (and their paths) of all files in the 'inpath' dir.
-> Depending on bool:cpluscplus, use either getdict() in dictbuilder.py or just use simple bag-of-words to build the dict.
-> (if bool:stub is TRUE) Find the word-freq-dict for the stub code and subtract this from all other file's corresponding dictionaries.
-> Calculate the percentage similarity for each pair of files and store in a numpy array.
-> Using this data, an appropiate .csv file and a heatmap image is plotted using matplotlib; and saved in the 'outpath' dir.
The csv file has "File A", "File B" and "Plagiarism Percentage" as columns (values in "Plagiarism Percentage" column being descendingly sorted).
